크롬 확장 프로그램 압축버전입니다.
크롬 확장프로그램 스토어의 검토가 끝난 경우 해당 링크를 공유드리겠습니다.


# 설치방법 
1. ZIP 다운로드 후 압축 해제
2. chrome://extensions/ 접속
3. 우측 상단의 개발자 모드 활성화 (우측 상단 토글)
<img width="1529" height="323" alt="image" src="https://github.com/user-attachments/assets/1d8c2f9f-011e-4129-a428-c044edbe04ab" />

4. 압축해제된 확장 프로그램을 로드합니다
- '압축 해제된 프로그램로드' 버튼 클릭 ▶ 압축해제 폴더 선택

5. 크롬 우측 상단의 추가된 프로그램 확인

<img width="388" height="307" alt="image" src="https://github.com/user-attachments/assets/4ca39c54-5c7a-47e8-9414-4a1067a896e6" />
